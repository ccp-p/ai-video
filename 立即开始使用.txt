========================================
 3步立即使用
========================================

第1步：安装FFmpeg（如未安装）
  网址：https://ffmpeg.org/download.html
  下载后解压到 C:\ffmpeg 并添加到系统PATH

第2步：启动服务
  方法A：双击 start.bat → 选择 1
  方法B：命令行运行：ccode_enhanced.exe -mode server -port 8080

第3步：打开浏览器
  访问：http://localhost:8080
  开始使用！


========================================
 3大功能
========================================

1. 视频处理
   - 输入视频路径
   - 自动生成字幕(SRT)
   - 提取视频截图
   - 导出音频文件

2. AI总结
   - 上传SRT或TXT文件
   - 输入文本进行总结
   - 自定义Prompt
   - 获取Markdown格式报告

3. AI配置
   - 配置外部AI API
   - 支持OpenAI、文心一言等
   - 保存配置状态
   - 不配置也能用本地算法

========================================
 故障排查
========================================

❌ 找不到ffmpeg？
   → 下载FFmpeg并添加到PATH环境变量

❌ 端口被占用？
   → 运行：ccode_enhanced.exe -mode server -port 9090

❌ 视频处理失败？
   → 检查文件路径是否正确
   → 确保视频格式支持(MP4/AVI/MKV)
   → 查看控制台错误信息

❌ 打不开网页？
   → 确保服务已经启动
   → 检查防火墙是否阻止8080端口
